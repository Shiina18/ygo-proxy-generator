# ygo-proxy-generator

遊戯王プロキシ印刷ツール。YDK デッキリストから印刷用カード画像 PDF を生成します。多言語カード画像と、中国語テキストを効果欄に上書きするオーバーレイに対応し、印刷時の文字をくっきり保ちます。

- **カード画像の言語**: 中国語 / 簡体字中国語 / 日本語 / 英語
- **効果テキストのオーバーレイ**（中国語カードのみ）: 効果欄を覆って中国語テキストを書き直し、印刷文字をくっきり保つ
- **カード間の余白調整**: 3×3 グリッドのカード間隔を調整可能

### 開発者ガイド

- **インストール & ローカル開発**

```bash
npm install
npm run dev
```

- **ビルド & プレビュー**

```bash
npm run build
npm run preview
```

- **テスト**

```bash
npm run test
```
