# 游戏王代牌打印工具

把 YDK 一键转成可打印的卡图 PDF，支持多语言卡图与中文效果覆盖，保证效果文本打印清晰。

- **多语言卡图**：中文 / 简中 / 日文 / 英文
  - 增殖的 G (中文) / 增殖的 Z (简中)
- **覆盖效果文本**（仅中文卡图）：覆盖效果区文字，保证打印清晰
- **支持调整卡片间距**

### 开发者指南

- **安装与本地开发**

```bash
npm install
npm run dev
```

- **构建与预览**

```bash
npm run build
npm run preview
```

- **测试**

```bash
npm run test
```
