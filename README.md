# ygo-proxy-generator

Yu-Gi-Oh! proxy printing tool. Turn a YDK deck list into a printable card-image PDF, with multi-language card images and a Chinese effect-text overlay that keeps printed text perfectly clear.

## Features

- **Card image languages**: Chinese, Simplified Chinese, Japanese, English
- **Effect-text overlay** (Chinese only): overwrite the effect box with Chinese text to guarantee sharp print quality
- **Adjustable spacing**: control spacing between cards in the 3×3 grid

## Developer guide

- **Install & dev**

```bash
npm install
npm run dev
```

- **Build & preview**

```bash
npm run build
npm run preview
```

- **Test**

```bash
npm run test
```
